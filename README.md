## Instalasi Project

Bagaimana cara menjalankan project ini? Ikuti langkah-langkah dibawah ini.

* pertama siapkan alat alat dulu 
    * ESP32
    * DHT11
    * Kabel 
* pada bagian wirring 
    * input DHT11 diletakan di `GPIO4`
    * VCC di pasangkan di `3.3v / 5v` (direkomendasikan 3.3v)
    * Ground ke Ground
* jika sudah clone branch ini
```bash
    git clone -b ghibli https://github.com/takathena/dht11-schoolwork.git
```
* jalankan `main.py` di software Thonny atau sejenisnya
* jika sudah dijalankan, akan muncul ip, misal `192.168.32.4`
    * click ip tersebut, lalu akan diarahkan ke browser
* Voila!! dashboard sudah muncul
